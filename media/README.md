# Media & Photos: Private Digital Asset Management

The homelab provides a high-performance, private media and photo ecosystem for the whole family — no third-party cloud storage required.

## 🖼️ Private Photo Management: Immich

**Immich** is a modern, open-source photo manager that replaces commercial cloud photo storage entirely.

- **Mobile Sync:** Automatic, encrypted photo and video backups from family mobile devices.
- **Smart Search:** Immich's built-in ML pipeline provides face recognition, object detection, and CLIP-based semantic search out of the box.
- **High Performance:** Hardware-accelerated transcoding for smooth browsing of large libraries.
- **Full Privacy:** No photos are ever uploaded to external servers.

## 📺 Private Entertainment: Plex

**Plex Media Server** provides a centralized, easy-to-use interface for streaming our private media collection.

- **Any Device Access:** Streams to smart TVs, mobile phones, and web browsers.
- **Auto-Transcoding:** Hardware acceleration for high-resolution video on lower-bandwidth connections.
- **Managed Library:** A fully searchable, metadata-rich library of private digital media.

## 🔄 The "-arr" Stack: Media Automation

To keep the media collection organized, the entire management lifecycle is automated using the **-arr** ecosystem.

- **Sonarr & Radarr:** Automated managers for TV shows and movies.
- **Overseerr:** A user-friendly request interface that allows family members to request new media.
- **Library Sync:** Automatically organizes, names, and moves files into the final storage vault.

## 🛡️ Data Redundancy & Protection

Every photo and media file is protected by the homelab's core storage strategy:
- **ZFS Mirroring:** Protection against individual drive failures.
- **Daily Backups:** Snapshots of the primary media storage pool.
- **Remote Archive:** Weekly encrypted backups of irreplaceable family photos to off-site storage.
